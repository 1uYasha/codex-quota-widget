# 更新日志

## 1.3.0 — 2026-07-14

- 软件正式更名为 **ChatGPT Quota**，同步更新窗口标题、菜单文案、包名、Bundle ID 与构建产物名称。
- 启用新的猫耳几何结图标，并统一应用到 Mac Dock/Finder/菜单栏、Windows EXE/任务栏/托盘和 README 展示。
- Mac 菜单栏新增“图标 + 剩余额度百分比”常驻状态项，小组件隐藏后仍可查看额度。
- Mac 菜单栏百分比可选择显示 7 天或 5 小时额度，选择结果自动保存。
- Mac 点击菜单栏图标可打开专属设置面板，支持显示/隐藏、刷新、开机启动、刷新间隔与退出。
- Mac 点击小组件关闭按钮时改为隐藏窗口，程序继续在菜单栏运行。
- Windows 保留托盘左键显示/隐藏与右键设置菜单，并继续提供免安装 x64 便携版。
- 新增 Mac 与 Windows 平台差异说明和对照图。
- Release 同时提供 Apple Silicon Mac ARM64 ZIP 与 Windows x64 Portable EXE；两个构建暂未签名。

## 1.2.0 — 2026-07-13

- 恢复 5 小时额度，并与 7 天额度、水位仪表分别提供显示开关。
- 水位仪表可选择跟随 5 小时或 7 天额度，设置会在本机自动保存。
- 水位低于 40% 时变为橙色，低于 20% 时变为红色。
- 增强便携版对 Codex Desktop、全局 npm/pnpm/Bun CLI 与 PATH 安装的发现和回退能力。
- 延长 Codex 冷启动等待时间，并在组件底部显示可行动的失败原因。
- 刷新时保持水位和界面颜色不变，仅在左下角显示蓝点与“正在刷新”；完成后恢复绿点和更新时间。
- 左上角读取状态固定显示“读取正常”或“读取失败”，不再随刷新切换为“读取中”。
- 窗口缩放时同步调整仪表、卡片、按钮、间距和字体，改善小尺寸拥挤与大尺寸留白。
- 标题栏精简为居中的 `···`、隐藏和退出三个按钮，刷新、置顶、语言及显示选项统一收进 `···` 菜单。
- 关闭水位显示后，窗口可缩小至 `180×140`，仅保留控件与文字正常显示所需空间。
- 新增 Apple Silicon（M 系列）Mac 打包与 Codex CLI 发现支持；Mac 端采用原生 ARM64 运行时，并避免后台节流以保持悬浮窗口响应速度。
- `v1.2.0` Release 同时提供 Windows x64 便携版和 Apple Silicon Mac ARM64 版本；Mac 版支持从 ChatGPT.app 内置路径读取 Codex CLI。

## 1.1.0 — 2026-07-13

- 移除已取消的 5 小时额度读取和显示，仅保留 7 天额度。
- 支持从 Codex 新旧接口响应中识别 7 天额度窗口。
- 支持从窗口边缘自由调整组件大小，并自动记住上次尺寸。
- 合并重复额度请求，并缓存今日 Token 日志统计，降低刷新开销。
- 修复额度请求写入竞态、异常输出无限增长和无效百分比误显示问题。
- 增加内容安全策略，禁止页面跳转和外部新窗口。
- 更新中英文文档、预览图和测试用例。
